Double Pendulum Model

The background integrates a gravity-free, undamped two-link pendulum from random initial conditions.

\[ \dot{\theta}_1 = \omega_1, \qquad \dot{\theta}_2 = \omega_2, \qquad \Delta = \theta_1 - \theta_2 \] \[ \dot{\omega}_1 = \frac{-2\sin(\Delta)m_2\left(\omega_2^2L_2 + \omega_1^2L_1\cos(\Delta)\right)}{D_1} \] \[ \dot{\omega}_2 = \frac{2\sin(\Delta)\left(\omega_1^2L_1(m_1 + m_2) + \omega_2^2L_2m_2\cos(\Delta)\right)}{D_2} \] \[ D_1 = L_1\left(2m_1 + m_2 - m_2\cos(2\Delta)\right) \] \[ D_2 = L_2\left(2m_1 + m_2 - m_2\cos(2\Delta)\right) \]
State
\(\theta_1, \theta_2, \omega_1, \omega_2\)
